# Northstar

Northstar is a mentorship platform focused on developing leaders through structured guidance, skill-building, and real-world experience. By connecting driven individuals with experienced mentors, Northstar helps members grow professionally, build discipline, and create meaningful impact.

## WordPress Theme Included

This repository now includes a high-level, production-ready WordPress theme scaffold at:

`wordpress/wp-content/themes/northstar`

### What it provides

- 5 primary navigation tabs: **Home**, **Mentoring**, **Donate**, **Contact**, and **Northstar Store** (external Shopify link).
- Starry-night visual language inspired by the Northstar concept.
- Interactive sparkle animation for the hero section.
- Sales-focused nonprofit messaging for lead capture and CTA performance.
- Secure and extensible PHP theme files with escaping and WordPress best practices.
- SEO-minded metadata and performance-friendly structure.

### Quick start

1. Copy the `northstar` folder into your WordPress installation under `wp-content/themes/`.
2. Activate the theme in WordPress Admin → Appearance → Themes.
3. In the Customizer, set your **Shopify URL** under **Northstar External Links**.
4. Connect your preferred donation processor and form plugin for live submissions.

### Recommended plugins for production hardening

- Wordfence or Sucuri (WAF/security)
- WP Rocket or LiteSpeed Cache (performance)
- Rank Math or Yoast SEO (advanced SEO)
- WPForms or Gravity Forms (contact + automation)
